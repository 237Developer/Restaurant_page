# Restaurant Page

The initial goal of this project was not to build a restaurant website, but rather to learn how to use a bundler, specifically Vite. However, I am quite pleased with how the final result turned out!  🙂

## Live Demo

You can visit the live website here: [demo](https://237developer.github.io/Restaurant_page/)

## How to Install and Run

1. Clone the repository:
   ```bash
   git clone https://github.com/237Developer/Restaurant_page.git
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the local development server:
   ```bash
   npm run dev
   ```
4. Open your browser and navigate to the local address displayed in your terminal.

## What I Learned

- Configuring and setting up Vite from scratch.
- Integrating and managing Vite plugins.
- Working efficiently with npm and the `package.json` file.
- Implementing best practices for deployment on GitHub.
